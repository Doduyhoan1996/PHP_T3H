Vòng lặp while

		do{
		// code block to be executed
		// code thực thi
	}while(điều_kiện)

Vòng lặp while chỉ chạy khi điều kiện trong ngoặc đúng

Vong lặp do... while sẽ chạy khi điều kiện trong ngoặc đúng nhưng khi điều kiện không đúng thì nó chỉ chạy một làn đầu tiên duy nhất
