CSS Clearfix

Khi 1 thẻ html cao hơn thẻ mẹ chứa nó bà dc float left hoặc float right
=> Thì nó sẽ thị trượt khỏi thẻ
=> Thẻ mẹ không còn chứa được thẻ con
Vậy để giải quyết vấn đề này có 2 cách phổ biến
Cách 1: đặt thẻ mẹ thuộc tính overflow:hiden ( cách này có nhược điểm
Cách 2: đó là sử dụng class clearfix và đặt class clearfix này cho thẻ mẹ
.clearfix{
overflow:auto;
}
.clearfix::affter{
conten:"";
clear:both;
display:table
}
